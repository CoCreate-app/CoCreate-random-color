CoCreate-Color

A simple HTML5, CSS and pure javascript component, to generate a random color.

Install 

npm install cocreate-color

Or you can use cdn link:

https://cdn.cocreate.app/CoCreate-color.min.js

Usage

This is Color reference

                                <div data-background_color='random'> Color random </div>

Attributes

  • background_color string required

    random
